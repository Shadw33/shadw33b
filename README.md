# Application

Internal development project.

## Development Notes

- Frontend runs on port 3000
- Backend runs on port 3001
- Environment variables required for operation
  - `VITE_RECAPTCHA_SIGNUP_SITE_KEY` (reCAPTCHA v2 checkbox for signup flow)
  - `RECAPTCHA_SIGNUP_SECRET_KEY` (server-side verification for signup protection)
  - `RECAPTCHA_SIGNUP_ALLOWED_HOSTNAMES` (optional, comma-separated hostnames permitted for signup tokens)
